Class Number sẽ có (Integer, Byte, Long, Float, Double, Short) chi sử dụng chúng họ thường gọi là kiểu dữ liệu này là tham chiếu, lớp, đối tượng,... nghe có vẻ hợp lý 😬😬😬 tiếp tới là int, float, double, byte, short, long, char,... j đó hay sử dụng họ gọi chúng là kiểu dữ liệu nguyên thủy. Và chúng được bao bọc rôn hơn là Integer bao thằng int, Long bao long, ... như lớn bao nhỏ. Hình dung về 1 dãy nhe: Number bao Integer bao int 😏. Vấn đề box và unbox như sau: khi họ sử dung kiểu dữ liệu tham chiếu Integer x = 476473 họ gọi nó là box int thành Integer thì đúng rùi số đang sử dụng thăng Integer mà nó lại bao thằng int mà, tiếp theo họ sử dụng thằng trên để cộng như bình thường lúc này họ nói là unbox 😌 cha làm đc thì con phải làm đc. Vậy tóm lại sử dụng Integer hay int, ... đều là kiểu số nguyên,... chỉ đơn giản phạm vi. Và tham chiếu và nguyên thủy sẽ khác nhau nó sẽ cang thiệp vào bộ nhớ như con trỏ ở C/C++
