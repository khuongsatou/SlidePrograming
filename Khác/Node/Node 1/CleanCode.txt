Clean code là code đã được "quy hoạch", sắp xếp, có cấu trúc rõ ràng, mạch lạc sao cho code đơn giản dễ đọc, dễ test, dễ bảo trì. Có thể liên tưởng clean code như sự quy hoạch hạ tầng đô thị, thành phố hiện đại, sạch đẹp với các khu vực, những phân khu chức năng riêng biệt được kết nối bởi mạng lưới giao thông mạch lạc, thông suốt không bị rối, chồng chéo. Ví dụ, Quy hoạch tp HN giống như Not Clean Code, còn tp New York như Clean Code
